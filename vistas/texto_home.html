En {% include 'texto_ganemos.html' %} hemos puesto en marcha un proceso para elaborar de forma colectiva un programa electoral con las necesidades y demandas de la ciudadnía.

El primer paso de este proceso es identificar cuáles son aquellas propuestas sobre las que existen un amplio consenso ciudadano. A través de esta web podemos recoger y evaluar de forma colaborativa cuál es el grado de acuerdo sobre diferentes propuestas. Las de más consenso sobre cada área se tomarán como la base sobre la que construir el programa de {% include 'texto_ganemos.html' %}. Se puede tener acceso informativo a todos los contenidos de esta web pero para entrar realizando propuestas programáticas, comentarios y/o valoraciones debes iniciar sesión o registrarte aquí.
