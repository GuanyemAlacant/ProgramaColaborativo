¿Para qué sirve esta web?

Esta web sirve para elaborar de manera colectiva el programa que {% include 'texto_ganemos.html' %} presentará en las próximas elecciones municipales.

Cualquier persona que se registre puede presentar aquí su propuesta, así cómo debatir, valorar comentar y sugerir mejoras a aquellas propuestas publicadas.

Esto significa que este foro es únicamente para debatir sobre el programa. Consulta la web de {% include 'texto_ganemos.html' %} para ver otros procesos de participación online abiertos.
